Git Commands And Their Explanation

  1. git init: initialize an existing directory as a Git repository
  2. git clone [url]: retrieve an entire repository from a hosted location via URL
  3. git clone [url]: retrieve an entire repository from a hosted location via URL git
  4. git status: show modified files in working directory, staged for your next commit
  5. git add [filename]: add a file as it looks now to your next commit (stage)
  6. git commit -m “message”: commit your staged content as a new commit snapshot
  7. git branch: list your branches. a * will appear next to the currently active branch
  8. git branch [branch-name]: create a new branch at the current commit
